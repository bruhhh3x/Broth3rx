## 🚀 DEPLOYMENT IN 3 STEPS

### Step 1: Download Files (2 minutes)
```
1. Download all files from /outputs/ folder
2. Organize into folder structure (see FILE-MANIFEST.md)
3. Keep folder structure exactly as:
   broth3r-website/
   ├── css/
   ├── js/
   ├── images/
   ├── index.html
   └── config files
```

### Step 2: Replace Placeholders (10-30 minutes)
```
1. Add images to /images/ folder:
   - Hero background image (1920×1080)
   - Hero video (30-60 sec, <10MB)
   - Event photos (3 images, 800×600+)
   - Logo files

2. Update links in index.html:
   - Singlelink URL
   - Bandcamp URL
   - Instagram profile
   - Contact email

3. (Optional) Integrate Event Schedule & Beehiiv
```

### Step 3: Deploy to Netlify (5 minutes)
```
Option A: GitHub Auto-Deploy (Recommended)
1. Push code to GitHub
2. Connect repo to Netlify
3. Done! Auto-deploys on every push

Option B: Direct Deploy
1. Drag & drop broth3r-website folder onto Netlify
2. Site lives instantly
```

**Total time to production: ~30-50 minutes** ⚡

### Typography (Ready to Use)
```
Display Font:    Space Grotesk (headings)

Sizing:          Fluid via clamp() — scales automatically
```

```
Consistent 4px, 8px, 16px, 24px, 32px, 64px, 128px scale

```

```
Entrance:        slideUp, fadeIn, scaleIn
```
